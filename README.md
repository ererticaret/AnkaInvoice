# 🧾 AnkaInvoice

**Simple and modern invoicing for freelancers and small businesses.**

AnkaInvoice is a lightweight web app that allows users to create professional invoices in seconds, manage clients, and keep track of payments — all in one simple interface.

---

## 🚀 Features

* Create invoices instantly
* Clean and minimal UI
* Fast and easy to use
* No accounting knowledge required

---

## 💡 Why AnkaInvoice?

Most invoicing tools are too complex or expensive.
AnkaInvoice focuses on simplicity and speed — helping you get paid without the hassle.

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

*(Future: Firebase, Stripe, Backend integration)*

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/ankainvoice.git
```

2. Open the project folder

3. Run `index.html` in your browser
